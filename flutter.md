# Flutter

## 概要
- 開発元はGoogle
- 2018にファーストリリース
- オープンソースのUI SDK
- 実はモバイルだけではない
	- Web, Windows, macOS, Linux

## 特徴
- 独自の描画エンジン
	- Impeller
	- Skia
	- ネイティブコンポーネントに依存しない
	- Flutter自身が直接描画
	- 60-120fpsと高速？
- ホットリロード
- 豊富なウィジェット
	- Flutterではすべての要素がウィジェットで管理される
	- iOS風、Android風のデザインを再現できる
	- オリジナルも

## 生まれた背景
- 異なるプラットフォームへの対応の手間
- 当時のWebベースのクロスプラットフォーム技術が遅かった
	- Apache Codova
	- React Native
- OS更新で崩れやすい
- など

## 歴史
- 2015 Skyとしてコンセプト発表
- 2018.12に 1.0リリース
